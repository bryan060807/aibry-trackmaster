## // SYSTEM OVERVIEW
**AIBRY TrackMaster** is a high-performance mastering environment built for the modern web. It combines a real-time Digital Signal Processing (DSP) engine with industrial-grade visual analysis and secure cloud synchronization.

Designed with a boutique hardware aesthetic, it allows engineers to finalize tracks with a professional signal chain—from harmonic saturation to a brickwall limiter—directly in the browser.

## // HARDWARE SPECIFICATIONS

### 🔊 Mastering Signal Flow
1. **Saturation:** Harmonic drive with wet/dry mix for analog warmth.
2. **Logarithmic EQ:** 3-band surgical shelf/peak filters.
3. **Spatial FX:** Convolution Reverb and Stereo Delay for depth.
4. **Compression:** Dynamics control with adjustable ratio and threshold.
5. **Output Gain:** Final makeup stage.
6. **Brickwall Limiter:** Zero-clipping safety ceiling fixed at **-0.1dB**.

### 📊 Visual Analysis Suite
* **Logarithmic Spectrum:** High-precision frequency readout (20Hz - 20kHz).
* **Spectrogram (Waterfall):** Time-based frequency heat map with Fire/Plasma palettes.
* **Oscilloscope:** Real-time waveform monitoring.

### ☁️ Cloud Integration
* **Mastering Logs:** Automatic cloud backup of mastered files via Supabase Storage.
* **Secure Auth:** OAuth2 (Google) handshake integration for private studio access.
* **Preset Manager:** Save and recall custom mastering "profiles" across sessions.
